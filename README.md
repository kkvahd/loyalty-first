# Loyalty First – Android App with Backend & Oracle DB 

## Overview 
Loyalty First is an Android application with a backend system designed to manage customer loyalty programs and transactions. The project focuses on integrating a mobile application with a backend service and a relational database. 

## Features 
- Android application for user interaction 
- Backend service for business logic 
- Oracle database integration for storing and managing data 
- CRUD operations for customer and transaction data 
- End-to-end data flow from UI → backend → database 

## Tech Stack 
- Android (Java) 
- Backend: Java with JDBC 
- Database: Oracle 

## What I worked on 
- Designed and implemented database interactions using Oracle 
- Built backend logic using JDBC for database connectivity 
- Implemented CRUD operations and query handling 
- Integrated Android app with backend services 
- Managed data flow between frontend, backend, and database 

## Project Structure 
- LoyaltyFirstAndroid/
– Android application - loyaltyfirstServer/ 
– Backend service
